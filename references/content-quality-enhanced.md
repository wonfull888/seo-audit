# 内容质量增强检查项 (3 项)

内容质量增强检查确保内容易读、专业且多媒体丰富。

---

## 1. 内容可扫描性

### CQ1 段落和句子长度分析

| 属性 | 值 |
|------|-----|
| 执行方式 | AI 分析文本结构 |
| 对应信号 | 用户体验和可读性 |
| 评分标准 | 平均段落≤5句,句子≤25词🟢 / 适中🟡 / 过长🔴 |

**原理**: 短段落、短句子和频繁使用列表提高内容可扫描性,用户能快速找到需要的信息。

**可扫描性指标:**
1. **段落长度**: 每段≤5句话
2. **句子长度**: 每句≤25个词(中文≤30字)
3. **列表使用**: 每500字至少1个列表
4. **小标题频率**: 每200-300字1个H2/H3

**好的示例:**
```
## 如何优化图片

优化图片可以显著提高页面加载速度。以下是3个关键步骤:

1. **压缩**: 使用TinyPNG等工具压缩
2. **格式**: 优先使用WebP格式  
3. **懒加载**: 非首屏图片添加loading="lazy"

每个步骤都能减少20-50%的文件大小。
```

**差的示例:**
```
图片优化是一个非常重要的性能优化手段,它可以通过压缩、格式转换、懒加载等多种方式来实现,具体来说,压缩可以使用各种在线工具或命令行工具,格式方面WebP是目前最推荐的选择,而懒加载则可以通过HTML属性或JavaScript来实现。
```

## 2. 术语定义

### CQ2 专业术语首次出现时的解释

| 属性 | 值 |
|------|-----|
| 执行方式 | AI 识别专业术语并检查是否有定义 |
| 对应信号 | 内容可理解性 |
| 评分标准 | 主要术语有定义🟢 / 部分有🟡 / 无🔴 |

**原理**: 定义专业术语让不同背景的读者都能理解内容,同时向搜索引擎明确说明概念。

**实现方式:**

1. **内联定义**:
```
SEO(搜索引擎优化)是提高网站在搜索结果中排名的技术。
```

2. **悬停提示**:
```html
<abbr title="搜索引擎优化">SEO</abbr>是...
```

3. **链接到词汇表**:
```html
<a href="/glossary#seo">SEO</a>是提高排名的关键。
```

## 3. 多媒体丰富度

### CQ3 视频/音频/交互式内容统计

| 属性 | 值 |
|------|-----|
| 执行方式 | HTML 解析 `<video>`, `<audio>`, `<iframe>` 等标签 |
| 对应信号 | 内容丰富度和用户停留时间 |
| 评分标准 | 有多种媒体类型🟢 / 仅图片🟡 / 无多媒体🔴 |

**原理**: 多样化的媒体内容提高用户参与度和停留时间,这是重要的排名信号。

**检测内容类型:**
- 📹 **视频**: `<video>`, YouTube/Vimeo嵌入
- 🎵 **音频**: `<audio>`, Podcast嵌入
- 📊 **交互式图表**: iframe嵌入的可视化工具
- 🎮 **交互式工具**: 计算器、配置器、测验

**评分标准:**
- 🟢 包含3+种媒体类型
- 🟡 仅有图片和视频
- 🔴 仅纯文本或仅图片

**示例:**
```html
<!-- 视频 -->
<iframe src="https://www.youtube.com/embed/..." ></iframe>

<!-- 音频 -->
<audio controls>
  <source src="podcast.mp3" type="audio/mpeg">
</audio>

<!-- 交互式工具 -->
<iframe src="https://calculator.example.com/seo-roi"></iframe>
```
