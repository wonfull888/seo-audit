# v1.4.1 任务清单 - 分类增强与可解释性

## 目标

在 v1.4.0 MVP 稳定基础上，提升站点分类准确度、可解释性和文章页命中率。

---

## 功能范围

1. 信号增强：`Title + URL` 升级为 `Title + URL + Nav`
2. 输出增强：Top-2 候选类型 + 置信度
3. 交互增强：低置信度快速确认
4. 文章页二次检索增强（常见内容路径）
5. 报告附录输出判定证据与路径
6. 指标化测试（准确率/命中率/回退成功率）

---

## 任务拆分

### Phase 1: 分类增强

- [ ] 增加导航关键词信号提取
- [ ] 调整分类权重并验证
- [ ] 输出 Top-2 结果

### Phase 2: 置信度与交互

- [ ] 定义置信度阈值
- [ ] 低置信度场景快速确认
- [ ] 无响应默认值策略

### Phase 3: 抓取增强

- [ ] 文章页二次检索（`/blog`、`/news`、`/article`、`/post`）
- [ ] 对单页站补充可解释回退信息

### Phase 4: 报告与验证

- [ ] 附录新增分类证据摘要
- [ ] 测试集覆盖 7+1 类型
- [ ] 统计分类与选页指标

---

## 验收标准

1. Top-2 + 置信度在报告和流程中可见。
2. 低置信度场景有明确确认机制。
3. 文章页命中率较 v1.4.0 提升。
4. 有可复现测试结果与指标记录。
