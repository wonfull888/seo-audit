# v1.4.1 任务清单（原子任务版）- 分类增强与可解释性

## 目标

在 v1.4.0 MVP 基础上，提升分类准确率、文章页命中率与报告可解释性。

---

## 原子任务（16 项）

### A. 分类信号与权重（5 项）

- [x] A01 定义 Nav 关键词词典（中英，按 7+1 类型）
- [x] A02 实现 Nav 关键词提取器（从主导航文本抽取）
- [x] A03 将 Nav 分值接入分类器（Title + URL + Nav）
- [x] A04 将分类权重参数配置化（可调，不写死）
- [x] A05 完成权重回归验证并记录基线对比

### B. Top-2 与置信度（4 项）

- [x] B01 输出 Top-2 候选类型数据结构
- [x] B02 实现置信度计算规则（0-1）
- [x] B03 实现置信度阈值配置（高/中/低）
- [x] B04 在流程与结果中展示 Top-2 + 置信度

### C. 交互与抓取增强（3 项）

- [x] C01 实现低置信度快速确认提示
- [x] C02 实现无响应默认策略（超时后继续）
- [x] C03 实现文章页二次检索流程（`/blog`、`/news`、`/article`、`/post`、`/insights`、`/docs`）

### D. 报告可解释性（2 项）

- [x] D01 在报告附录增加分类证据字段（Title 命中、URL 命中、Nav 命中）
- [x] D02 在报告附录增加判定路径与回退路径说明（中英文模板同步）

### E. 测试与验收（2 项）

- [x] E01 扩展 7+1 类型测试集并执行回归
- [x] E02 产出指标报告（分类准确率、选页命中率、回退成功率）

## 验收标准

1. 分类结果可输出 Top-2 + 置信度，并可解释。
2. 低置信度场景可确认，且无响应不阻断。
3. 文章页命中率较 v1.4.0 有可量化提升。
4. 提供可复现实验数据与指标报告。
