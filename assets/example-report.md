## 4. 优化建议

### 4.1 技术 SEO 优化

**高优先级：**

1. **优化首页 LCP (3.8s → <2.5s)**
   - 首屏大图从 1.2MB JPEG 转换为 WebP 格式（预计减少 60%）
   - 添加 `loading="eager"` 和 `fetchpriority="high"`
   - 考虑使用 CDN 加速图片加载

2. **修复文章页 CLS (0.32 → <0.1)**
   - 为文章中的图片预设宽高：`<img width="800" height="450" ...>`
   - 侧边栏广告位添加固定尺寸容器
   - 字体加载添加 `font-display: swap`

3. **添加缺失的 Canonical 标签**
   ```html
   <!-- 分类页添加 -->
   <link rel="canonical" href="https://brewmaster.coffee/products/coffee-beans" />
   ```

**中优先级：**

4. **添加 Breadcrumb 结构化数据**
   ```json
   {
     "@context": "https://schema.org",
     "@type": "BreadcrumbList",
     "itemListElement": [{
       "@type": "ListItem",
       "position": 1,
       "name": "首页",
       "item": "https://brewmaster.coffee/"
     }, {
       "@type": "ListItem",
       "position": 2,
       "name": "咖啡知识",
       "item": "https://brewmaster.coffee/blog"
     }]
   }
   ```

### 4.2 页面元素优化

   <!-- 当前 -->

### 4.3 内容与 E-E-A-T 优化

**高优先级：**

1. **增强作者实体信息**
   
   创建作者页面 `/authors/xiaoli`，包含：
   - 真实照片
   - 专业背景（SCA 认证、从业年限）

   - 发表的文章列表
   
   添加 Author Schema：
   ```json
   {

     "name": "咖啡师小李",

   ```

2. **添加第一人称真实体验**
   
   在文章中加入：
   - "我测试了 5 款不同价位的手冲壶..."
   - "在使用这款滤杯的三个月里..."
   - 附上实际使用的照片（带水印）
